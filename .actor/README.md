# Actor Folder Reference

This folder contains Apify-facing artifacts.

1. `actor.json`
- Actor metadata.
- Build and run command used by Apify.

2. `input_schema.json`
- Apify GUI input form schema (the file Apify uses for Input tab form rendering).

3. `output_schema.json`
- Standard JSON Schema for dataset output records (job records and RUN_SUMMARY record).

4. `sample_input.json`
- Ready-to-run sample payload for Apify runs.
